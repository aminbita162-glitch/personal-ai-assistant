# 🧠 Personal AI Assistant

A smart personal assistant app designed for busy people to manage tasks, notes, and daily schedules with the power of AI.

---

## 🚀 Features (MVP)

- ✅ Task Management (create, update, complete)
- 📅 Daily Dashboard
- 📝 Notes System
- 🤖 AI Assistant (powered by OpenAI)
- 🔐 User Authentication

---

## 🧩 Tech Stack

- Backend: Python
- Database: PostgreSQL
- AI: OpenAI API
- Version Control: GitHub
